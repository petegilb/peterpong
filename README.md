Pong implementation using C++ and [raylib](https://www.raylib.com/).
Created for C++ practice and to get familiar with raylib.

<h3>Play</h3>
Download executable from the GitHub releases and run it.

<h3>Controls</h3>
W and S Keys for Player 1 + Arrow Keys for Player 2.

<h3>Build</h3>
Build with CMAKE
